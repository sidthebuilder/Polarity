# Polarity ✨

A beautifully designed, fun frontend prototype project exploring UI/UX, glassmorphism, and 3D motion! 

This was created as a fun experiment to build an interactive "dilemma resolver" using React, Tailwind CSS, and Framer Motion (`motion/react`). It features a 3D card-flipping system and slick, atmospheric styling. 

## Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the dev server:
   ```bash
   npm run dev
   ```

*Note: This is currently a static frontend prototype and does not require any backend or API keys to run.*
